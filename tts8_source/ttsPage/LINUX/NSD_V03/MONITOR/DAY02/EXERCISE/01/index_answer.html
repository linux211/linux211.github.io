2 Cacti使用什么存储监控数据

参考答案

snmp和rrdtool是cacti的关键。Snmp关系着数据的收集，rrdtool关系着数据存储和图表的生成。

3 修改snmpd.conf配置文件

要求：实现让192.168.1.200监控主机有权过来监控本机数据资源。

参考答案

[root@client ~]# vi /etc/snmp/snmpd.conf
……
com2sec notConfigUser  192.168.1.200    public   //设置IP地质
access  notConfigGroup ""  any   noauth   exact  all none none  //赋予视图权限
view all    included  .1                          80             //创建视图
……
[root@localhost ~]# service snmpd start
