1 循环语句的else的执行条件是什么

参考答案

循环语句while或for尾部的else只有在循环体正常结束时才会执行，如果循环体内执行了break语句，那么else子句也会被中断，不被执行。

2 编写九九乘法表

要求: 编写一个脚本，该脚本用于打印九九乘法表。然后进一步完善，可以打印出任意基数的乘法表。

参考答案

本题需要使用嵌套循环的方式，外层循环用于决定打印多少行，内层循环用于决定该行打印多少项。内层循环打印的项目数与外层是第几次循环一致。

只打印九九乘法表可以使用如下方法：

[root@py01 bin]# vim multable.py

#!/usr/bin/env python

for i in range(1, 10):
    for j in range(1, i + 1):
        print "%d X %d = %d" % (j, i, i * j),    #加上逗号抵制print自动加上的回车
    print               #每行打印结束，再打印一个回车

为了可以打印任意基数的乘法表，并且可以方便的实现代码重用，修改优化后的程序代码如下：

[root@py01 bin]# vim multable2.py

#!/usr/bin/env python

def tables(num = 9):
    for i in range(1,num+1):
        for j in range(1,i+1):
            print "%d * %d = %d " % (j,i,i*j),
        print

if __name__ == '__main__':
    tables()
    tables(6)

3 以写入方式打开文件的注意事项

当打开文件的时候，指定模式为’w’，要注意什么？

参考答案

需要注意的是，如果指定的文件不存在，会创建文件；如果文件已经存在，则会清空文件。

4 提供正确的函数返回值

下面代码执行后，sum10的返回值是什么

>>> def myadd(x, y):
...     print x + y
... 
>>> sum10 = myadd(10, 5)

参考答案

返回值为None。函数的返回值由return语句提供，如果没有明确的return语句，默认返回值为None。
