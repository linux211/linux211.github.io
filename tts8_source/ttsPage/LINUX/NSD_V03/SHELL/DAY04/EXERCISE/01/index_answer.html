2 编写whami.sh脚本，检测当前脚本放在哪个目录下、文件名是什么。

参考答案

1）编写whami.sh脚本文件

[root@svr5 ~]# cat whami.sh 
#!/bin/bash
cd $(dirname $0) && pwd
basename $0

[root@svr5 ~]# chmod +x whami.sh

2）测试、验证执行效果

[root@svr5 ~]# ./whami.sh 							//正常识别路径、文档名
/root
whami.sh

[root@svr5 ~]# cp whami.sh /opt/new.sh
[root@svr5 ~]# /opt/new.sh  						//换位置、改名后一样可识别
/opt
new.sh

3 简述egrep工具的-q选项的含义，并验证其在脚本应用中的价值。

参考答案

选项-q的作用是静默、无任何输出，效果类似于正常的egrep操作添加了&> /dev/null来屏蔽输出。在脚本中当需要检测文本内是否存在指定的关键词时，可以利用egrep -q的返回状态值来作为判断依据，从而简化&>输出的操作。

验证效果可参考：

[root@svr5 ~]# cat /etc/shells 							//查看测试文本
/bin/sh
/bin/bash
/sbin/nologin
/bin/dash
/bin/tcsh
/bin/csh
/bin/zsh 

[root@svr5 ~]# egrep -q '/zsh' /etc/shells && echo yes || echo no
yes  														//支持zsh
[root@svr5 ~]# egrep -q '/bsh' /etc/shells && echo yes || echo no 
no 														//不支持bsh

4 正则表达式中的+、？、*分别表示什么含义？

参考答案

这三个字符用来限制关键词的匹配次数，含义分别如下：

  • +：最少匹配一次，比如a+可匹配a、aa、aaa等
  • ？：最多匹配一次，比如a?可匹配零个或一个a
  • *：匹配任意多次，比如a*可匹配零个或任意多个连续的a

5 如何编写正则表达式匹配11位的手机号？

参考答案

准备测试文件：

[root@svr5 ~]# cat tel.txt 
01012315
137012345678
13401234567
10086
18966677788

提取包含11位手机号的行：

[root@svr5 ~]# egrep '^1[0-9]{10}$' tel.txt 
13401234567
18966677788
