1 配置httpd网站服务时，以下（ ）语句用来指定网站的根路径

A. DocumentRoot

B. DirectoryIndex

C. ServerName

D. Include

参考答案

正确答案为A，DocumentRoot可以定义Web页面的根路径。

2 配置httpd虚拟主机

要求：客户端输入URL：www.baidu.com和www.sina.com，可以分别访问到服务器上/web/baidu以及/web/sina目录下的首页文件。www.baidu.com站点首页内容为baidu,www.sina.com站点首页内容为sina。

参考答案

[root@svr5 ~]# mkdir –p /web/{baidu,sina}
vim  /etc/httpd/conf/httpd.conf
NameVirtualHost 192.168.4.5  	//服务的IP地址
<VirtualHost 192.168.4.5>  	//虚拟站点1
    DocumentRoot /web/baidu
    ServerName  www.baidu.com
</VirtualHost> 
<VirtualHost 192.168.4.5>  	//虚拟站点2
    DocumentRoot /web/sina
    ServerName  www.sina.com
</VirtualHost>

4 创建MySQL数据库及账户

要求：安装MySQL数据库软件，创建名为mytest的数据库，并创建名为maria的本地数据库账户，密码设置为11qq22，使得该账户对mytest数据库具有所有权限。

参考答案

[root@svr5 ~]# yum –y install mysql-server
[root@svr5 ~]# mysql  -u  root  -p
Enter password:  					//验证管理密码
mysql> CREATE  DATABASE  mytest;
mysql> GRANT  all  ON  mytest.*  TO  maria@localhost 
    IDENTIFIED  BY  '11qq22';
mysql> QUIT

5 简述部署Discuz!论坛的工作流程

参考答案

1）部署LAMP软件并启动相应服务；

2）创建网站数据库及数据库管理员账户及密码；

3）将Discuz！压缩包中的upload下的内容解压到httpd站点根路径下，并调整好适当的权限；

4）最后，在浏览器输入网址后，进行第一次的初始化工作。
