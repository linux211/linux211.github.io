1 以下（ ）命令可以查看所有的KVM虚拟机列表

A. virsh list

B. virsh list --all

C. virsh list -a

D. virsh list --any

参考答案

B选项,virsh list –all可以查看所有KVM虚拟机列表。

2 KVM虚拟机的XML描述文件主要包括哪些内容

参考答案

记录每个虚拟机的详细参数。如：名称、UUID、内存、 CPU、磁盘、……

XML文件的默认位置：/etc/libvirt/qemu/虚拟机名.xml

3 如何通过XML描述文件来新建一台虚拟机

参考答案

[root@svr5 ~]# virsh  dumpxml  已存在虚拟机名称  >  new.xml	//备份XML文件
[root@svr5 ~]# vim  new.xml					//修改名称、UUID、磁盘及MAC信息
<domain type='kvm' id='1'>
  <name>新虚拟机名称</name>  		        		//改名称
  <uuid>c33a4aad-c555-48b3-a0a9-893ad0b4775e</uuid>       //改UUID
    <disk type='file' device='disk'>
      <driver name='qemu' type='raw' cache='none'/>
      <source file=‘/data/images/node1.img’/>  	//指定磁盘
      <mac address=‘52:54:00:56:77:88’/>  		//指定MAC
      .. ..
    </disk>
.. ..  
[root@svr5 ~]# virsh  list  --all

4 使用virt-install从命令行安装KVM虚拟机

新建一台KVM虚拟机，要求如下：名称为node1，CPU设置为1个，内存为1024M，虚拟磁盘路径为/data/images/node2.qcow2，可以通过PXE网络来安装操作系统。

参考答案

[root@svr5 ~]# virt-install  --name node1  --vcpus=1  --ram=1024  \
--disk  path=/data/images/node2.qcow2,format=qcow2 \
--network  network=network1   --pxe
