1 邮件通讯中软件的角色有哪些

参考答案

Mail User Agent，邮件用户代理

Mail Transfer Agent，邮件传输代理

Mail Delivery Agent，邮件分发代理

A. 45

C. 25

D. 53

3 使用Postfix配置SMTP服务器

要求：Postfix服务器IP地址为192.168.4.5，服务器主机名为mail.tedu.cn，主邮件域为tedu.cn，默认发件域为tedu.cn，默认收件域为tedu.cn。

参考答案

[root@svr5 ~]# vim  /etc/postfix/main.cf
.. ..
inet_interfaces = 192.168.4.5  		
myhostname = mail.tedu.cn
mydomain = tedu.cn  							//主要邮件域
myorigin = $mydomain  								//默认的发件域
mydestination = $mydomain, $myhostname, localhost 	//本地管辖的投递域

4 Postfix配置SMTP认证需要修改哪些指令

参考答案

[root@svr5 ~]# vim  /etc/postfix/main.cf
.. ..
mynetworks = 127.0.0.1							//设置信任网络
smtpd_sasl_auth_enable = yes						//启用SASL认证
smtpd_sasl_security_options = noanonymous			//阻止匿名发信
smtpd_recipient_restrictions =						//设置收件人过滤
  permit_mynetworks,  
  permit_sasl_authenticated,
  reject_unauth_destination						//拒绝向未授权的目标域发信
[root@svr5 ~]# service  postfix  restart
