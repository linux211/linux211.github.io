1 使用keystone创建用户与租户

创建名为user2的用户，密码为redhat，同时创建一个名为proj1的租户。

参考答案

[root@openstack ~]# keystone user-create \
>--name user2 --pass redhat								//创建用户
[root@openstack ~]# keystone user-list						//列出用户
[root@openstack ~]# keystone tenant-create --name proj1		//创建租户
[root@openstack ~]# keystone tenant-list					//列出租户

2 Swift对象存储中Ring的作用是什么？

参考答案

Ring是Swift中最重要的组件，用于记录存储对象与物理位置间映射关系，Ring用来确定数据驻留在集群中的位置，有单独对应于Account、container和object的ring，Ring是存储在硬盘上的实体名称和物理位置间的映射，环使用区域、设备、分区和副本来维护这些映射信息

3 Openstack项目中Cinder的作用是什么

参考答案

Openstack从Folsom版本开始使用Cinder替换原来的Nova-Volume服务，为Openstack云平台提供块存储服务，Cinder可以为虚拟机 （VM）提供了持久块存储。

4 使用Cinder管理逻辑卷

使用Cinder创建和删除逻辑卷的命令分别是什么？

参考答案

[root@openstack ~]# cinder create  --display-name  vol1  2		//创建逻辑卷
[root@openstack ~]# cinder delete vol1							//删除逻辑卷
